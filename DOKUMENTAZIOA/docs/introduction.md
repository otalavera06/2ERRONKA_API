# Sarrera

Jatetxe kudeaketa sistemarako APIa da hau. Bertan, eskaerak, mahaiak, produktuak eta kategoriak kudeatzeko beharrezkoak diren endpoints-ak definitzen dira.

## Helburuak

- **Eskaeren kudeaketa**: Eskaerak sortu, eguneratu eta ordainketa egoera kontrolatu.
- **Mahaien kontrola**: Mahai libreak eta okupatuta daudenak kudeatu.
- **Produktuen zerrendak**: Produktuak kategoriaren arabera lortu eta stock-a eguneratu.
- **Segurtasuna**: Login sistema bidezko autentifikazioa eta log-en erregistroa.

## Erabilitako Teknologiak

- **Backend**: .NET 8 Web API
- **ORM**: NHibernate eta FluentNHibernate
- **Database**: MySQL
- **Dokumentazioa**: DocFX
